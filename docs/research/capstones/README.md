# Research Capstones

This directory holds the final research proposals and supporting artifacts for the research layer.

Use one subdirectory per capstone. Keep each capstone self-contained:

- `proposal.md` for the research question and falsifiable claim
- `experiment_plan.md` for the run plan and metrics
- `results.md` for the outcome and interpretation
- `figures/` for static outputs
- `notebooks/` for reproducible notebook artifacts when needed
- `reproduction.md` for exact rerun instructions

Do not let a capstone grow into a general dumping ground. The point of the directory is to keep one bounded research story readable from proposal through result.
