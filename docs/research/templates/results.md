# Results Template

## Summary

State the conclusion in one paragraph.

## Hypothesis

Repeat the original hypothesis.

## Baseline

Repeat the baseline used for comparison.

## Metric

Report the metric, unit, and aggregation rule.

## Ablation

Report the ablation and what changed.

## Assumptions

State which assumptions held and which did not.

## Risks

State the remaining confounders and interpretive limits.

## Failure Criteria

State whether the failure criteria were met and why.

## References

List the registry keys used in the experiment.
