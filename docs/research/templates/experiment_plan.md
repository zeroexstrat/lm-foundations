# Experiment Plan Template

## Question

What is being tested?

## Hypothesis

Write the hypothesis in a single sentence.

## Baseline

State the baseline implementation and why it is appropriate.

## Metric

State the metric, its unit, and how it will be aggregated.

## Ablation

Describe one ablation and the expected direction of change.

## Assumptions

List the assumptions that must hold for the experiment to mean anything.

## Risks

List the implementation, data, and interpretation risks.

## Failure Criteria

State the threshold for calling the experiment unsuccessful.
