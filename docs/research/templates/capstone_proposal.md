# Capstone Proposal Template

## Problem

Describe the research question in one paragraph.

## Hypothesis

State the hypothesis as a testable claim.

## Baseline

Name the strongest simple baseline.

## Metric

Name the primary metric and the unit of comparison.

## Ablation

Describe one ablation that should isolate the mechanism.

## Assumptions

List the assumptions required for the experiment to be interpretable.

## Risks

List the main confounders or failure modes.

## Failure Criteria

State the result that would falsify the proposal.

## References

List only registry-backed sources.
